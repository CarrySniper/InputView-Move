# InputView-Move
如何让输入框弹出键盘时自动上移。<br>
当键盘弹出，如果挡不到输入框，那么就不移动；<br>
如果挡到输入框，则会抬起到合适的位置显示输入框。<br>

百度经验：http://jingyan.baidu.com/article/fa4125acd3dfe228ad709263.html
